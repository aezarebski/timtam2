#+title: TimTam

This repository is an implementation of the [[https://github.com/aezarebski/timtam][TimTam]] likelihood (described in the
PLOS Computational Biology [[https://doi.org/10.1371/journal.pcbi.1009805][paper]]) as a BEAST2 package. See the [[https://aezarebski.github.io/timtam/index.html][webpage]] for
further information.
